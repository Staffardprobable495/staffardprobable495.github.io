## 📌 What is this tool?

The fleet-cve-scanner helps you manage computer security. It checks the software installed on your computers against known security flaws. Security teams track these flaws using IDs called CVEs. This tool reviews your software list and tells you which programs have bugs, how risky those bugs are, and if hackers currently use them to attack systems.

You do not need to be a security expert to use this tool. It gathers information from industry databases like the National Vulnerability Database and CISA. It then matches that information with the software found on your computers.

## 📋 System Requirements

To run this tool, your computer needs the following:

- Windows 10 or Windows 11.
- PowerShell 7. If you do not have it, the tool guides you through the installation.
- An RMM agent installed on the machines you want to scan.
- A basic internet connection to fetch the latest security updates.
- Standard user access rights or administrative rights depending on your RMM setup.

## 🔍 Understanding Security Data

The tool uses several specific indicators to grade your risk:

- **CVE:** This stands for Common Vulnerabilities and Exposures. It acts as a unique ID for a discovered security flaw.
- **EPSS:** The Exploit Prediction Scoring System estimates the probability that a hacker will use a flaw in the next 30 days.
- **CISA KEV:** This list contains vulnerabilities that attackers currently exploit. If a vulnerability appears here, act quickly.
- **SSVC:** This helps users decide how fast they must act based on the nature of the software and the flaw.

## 🛡️ Best Practices for Vulnerability Management

Running this tool once helps, but security requires consistent effort. We recommend a few simple habits:

- **Schedule scans:** Run this tool once a week to catch new vulnerabilities as they become public.
- **Update software:** When the report marks an item as critical, check your software supplier for an update patch.
- **Remove unused items:** If the scanner finds an old program that you no longer use, delete it. This reduces your surface area for potential attacks.
- **Keep the script updated:** Security data changes daily. Download the latest version of this tool whenever the developer releases an update to ensure you have fresh data.

## 🛠️ Troubleshooting

If you encounter errors, check these common items:

- **Execution Policy:** Windows occasionally blocks scripts. If you see an error involving execution policy, open PowerShell as an administrator and type: `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`.
- **Internet connectivity:** The scanner must connect to the internet to fetch vulnerability data. Ensure your firewall allows the script to reach these public security databases.
- **Inventory Access:** If the scanner reports zero results, check that your RMM software has exported your asset list correctly. The script reads files generated by your RMM system.

## 📝 Frequently Asked Questions

**Does this tool change my computer settings?**
No. This tool only reads information. It does not install anything or change your system configuration.

**Is my data sent to the internet?**
The tool sends requests to public databases to compare your software against known lists. It does not send your personal files or specific machine identifiers to private servers.

**What if I find a high-risk item?**
Prioritize updating the software listed. Check the website of the software provider for a "patch" or "security update."
